# 形态贴图配置数据模型

## 初始 FormResourceIndex 草案

```ts
interface FormResourceIndex {
  forms: FormResourceEntry[];
}

interface FormResourceEntry {
  formId: string;
  displayName?: string;
  textures: FormTextureSet;
  masks: FormMaskSet;
  models?: FormModelReference[];
  warnings?: FormResourceWarning[];
}

interface FormTextureSet {
  base?: string;
  overlay?: string;
  colormask?: string;
  overlayColormask?: string;
}

interface FormMaskSet {
  red?: string;
  green?: string;
  blue?: string;
  packed?: string;
}

interface FormModelReference {
  type: "bbmodel" | "geo" | "minecraft_model" | "unknown";
  path: string;
}

interface FormResourceWarning {
  code: string;
  message: string;
  path?: string;
}
```

## 路径原则

- 内部存储使用绝对路径或 workspace-relative path 需要统一选择。
- UI 展示可以显示相对路径。
- 写入操作必须解析到目标工作区内。

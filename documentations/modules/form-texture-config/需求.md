# 形态贴图配置模块需求

## 目标

提供可视化方式管理形态贴图、colormask、mask 输出和资源路径，减少手工处理资源包文件的成本。

## 基础需求

- 扫描目标模组中的形态贴图资源。
- 按 form 分组展示贴图。
- 展示 base texture、overlay、colormask 等资源。
- 支持配置 mask 处理输入输出。
- 支持预览处理结果。
- 支持生成或更新资源包结构。

## 首批建议支持内容

- 识别 `textures/form/<form_id>/` 下的贴图。
- 识别 `*_colormask.png` 和 `*_overlay_colormask.png`。
- 识别 `3d_models/player_form/<form_id>/colormask/` 下的 RGB mask。
- 调用或复用现有 mask 处理工具前，先建立输入输出说明。

## 验收标准

- 能列出已有 form 贴图资源。
- 能预览单个 form 的相关贴图。
- 能识别缺失的 colormask 或 overlay。
- 能在不破坏原文件的前提下生成处理结果。

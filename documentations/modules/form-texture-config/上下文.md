# 形态贴图配置模块上下文

## 模块定位

形态贴图配置模块负责编辑和管理形态相关的贴图、colormask、mask 输出、资源路径和后续打包流程。

该模块服务于形态资源制作流程，目标是减少手工处理贴图、mask 和资源包结构的重复劳动。

## 当前状态

目标模组仓库中存在大量形态贴图资源，典型位置包括：

- `src/main/resources/assets/shape-shifter-curse/textures/form`
- `3d_models/player_form`
- `custom_form_pack_example`

已有工具脚本包括：

- `tools/rgb_colormask_maker.py`
- `tools/MaskProcess.py`
- `tools/MaskProcessV2`
- `tools/MaskProcessV2_C`

## 模块边界

负责：

- 形态贴图资源发现
- 基础预览
- colormask 配置
- mask 处理流程配置
- 资源路径和输出路径管理
- 后续资源包/数据包打包入口

不负责：

- Power JSON 蓝图编辑
- 模组运行时代码
- 复杂 3D 模型编辑

## 当前风险

- 贴图、模型、动画、资源包路径较分散，需要先建立资源索引。
- 现有 Python/C 工具需要确认输入输出契约，避免 UI 与脚本耦合过深。
- mask 处理可能涉及二进制图片结果，必须有预览和回退机制。

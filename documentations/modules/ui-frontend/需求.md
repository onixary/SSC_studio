# 界面前端模块需求

## 目标

提供一个稳定、清晰、适合长时间配置工作的本地编辑器界面。

## 基础需求

- 支持打开/选择 `shape-shifter-curse-fabric` 项目目录。
- 展示工作区文件树。
- 支持模块入口导航。
- 支持编辑区标签页。
- 支持错误、警告和验证结果展示。
- 支持通用设置页。

## 交互要求

- 工具型界面应保持信息密度和可扫描性。
- 不做营销式首页。
- 默认第一屏应进入实际工作区或项目选择。
- 复杂编辑器区域应支持可调整面板尺寸。

## 验收标准

- 应用能启动到本地桌面窗口。
- 能选择或配置目标模组项目路径。
- 能进入三个主要模块入口。
- 前端基础布局在常见桌面分辨率下不出现明显重叠或空白。
